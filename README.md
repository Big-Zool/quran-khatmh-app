# Quran Khatm Platform (منصة ختم القرآن)

A collaborative web application to organize and track collective Quran recitations (Khatm) for family and friends.

## Features

- **Create Khatm**: Easily create a new Khatm session with a custom name and intention.
- **Share & Join**: Share a unique link with others to join the Khatm.
- **Page Reservation**: Participants can select specific pages to read.
- **Real-time Progress**: Track the overall progress of the Khatm in real-time.
    - **Remaining Pages**: See exactly how many pages are left to complete the Khatm.
- **Digital Tasbih (المسبحة الرقمية)**:
    - Integrated digital counter for Dhikr.
    - Floating access button on reading and share pages.
    - Persists counts per Khatm session.
    - Tracks "Total Clicks" and "Current Dhikr" automtically.
- **Multi-language Support**: Available in Arabic, English, Turkish, Urdu, and Farsi.
- **Dark Mode**: Fully supported comfortable reading experience.

## Tech Stack

- React + TypeScript + Vite
- Firebase Firestore (Real-time Database)
- TailwindCSS

## Development

1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build for production: `npm run build`
